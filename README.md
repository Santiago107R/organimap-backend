1. Instalar las dependencias 
```
yarn
```

2. Copiar __.env.template__, renombrar a __.env__ y cambiar las variables de entorno

3. Levantar Docker con la DB
```
docker compose up -d
```

4. Iniciar app
```
yarn start:dev
```
